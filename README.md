# Práctica 5 - Geomorfología Fluvial y Morfometría de Cuencas

**Autora:** Ruth Philippe  
**Materia:** Geomorfología (UASD)  
**Ubicación de la cuenca:** San Cristóbal, República Dominicana  

Este proyecto incluye el análisis morfométrico de una cuenca hidrográfica, aplicando los conceptos de geomorfología fluvial de Gutiérrez Elorza (2008) y De Pedraza (1996).  

## Contenido
- `Practica05.ipynb`: Notebook con todos los cálculos y análisis.

## Resultados destacados
- Densidad de drenaje: **1.50 km/km²**
- Razón de bifurcación promedio: **2.49**
- Orden máximo de red: **4**
- \( R_b^{reg} = 2.45 \) (según ley de Horton)
